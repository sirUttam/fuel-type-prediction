 
# ⛽ Fuel Type Prediction ML Project

This is a machine learning project that predicts the fuel type of a vehicle (Diesel or Petrol) based on:

- Mileage
- Engine size
- Power

---------------------------------------

## 🧠 Model Used
- Random Forest Classifier

-------------------------------------------

## 📊 Workflow
1. Data preprocessing
2. Label encoding (Fuel Type)
3. Feature scaling
4. Model training
5. Saving model using joblib
6. Streamlit web app for prediction

-----------------------------------------------

🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

-------------------------------------------

📌 Output
Predicts:

- Diesel
- Petrol

----------------------------------------------

## 🚀 How to Run

```bash
streamlit run app.py
